Retail Sales Capstone Project
Author: Shridevi Kulkarni

 Summary
  Analysis of Walmart retail sales to identify key drivers and build a baseline prediction model.

 Data
	•	train.csv – Weekly sales
	•	features.csv – Holidays & economic indicators
	•	stores.csv – Store type & size

 Key Insights
	•	Holidays significantly boost sales (T-test).
	•	Store types show statistically different sales (ANOVA).
	•	Type A stores perform best.
	•	Economic indicators show weak correlation.

 Model
  Linear Regression
	•	R²: 0.09
	•	MAE: 14,570
	•	RMSE: 21,767

 Files
  Notebook, PPT, PDF, and datasets included.

 Tools
  Python, Pandas, NumPy, SciPy, Sklearn, Matplotlib, Colab.
